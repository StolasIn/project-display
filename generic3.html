Linear algebra

  1. 矩陣名詞 :

    • sequare matrix (方陣) : 行數=列數的矩陣 n*n階的方陣
    • unit matrix (單位矩陣) : 對角線元素為1其餘為0的方陣
    • zero matrix (0矩陣) : 所有元素皆為0的矩陣
    • matrix transpose (轉置矩陣) : 矩陣行列互換 記為 A^T
    • matrix inverse (反矩陣) : 記為A^-1 滿足A*(A^-1)=(A^-1)*A=I
    • diagonal matrix (對角線矩陣) : 除主對角線元素外其他元素皆為0的矩陣
    • triangular matrix (上/下三角矩陣) : 所有主對角線下/上的元素皆為0
    • symmtric matrix (對稱矩陣) : 主對角線以外的元素可經由主對角線兩兩對稱
    • singular/nunsingular matrix (奇異/非奇異矩陣) : 該矩陣沒有/有矩陣反元素存在
    • elementary matrix (列操作矩陣) : 一個單位矩陣經過一次的列運算



  2. matrix properties :

    • 矩陣不一定滿足指數律以及乘法的交換律
    • A*B=A*C 但 B不一定等於C
    • (A*B)^T = (B^T)*(A^T) 且 (A*B)^-1 = (B^-1)*(A^-1)
    • ut:上三角矩陣 lt:下三角矩陣 : (ut/lt)^T 還是ut/lt , ut*ut 或 lt*lt 都依然是ut/lt
    • 假設A是symmtric matrix : A^T=A
    • 對角線矩陣 (diagonal matrix) 的矩陣反元素是所有元素取倒數(自乘n次則是把所有元素取n次方)
    • (k*A)^-1 = (A^-1)/k k是一個常數
    • (A^n)^T = (A^T)^n
    • 所有elementary matrix都是可逆的且他的反矩陣也是elementary matrix
    • 如果一個矩陣可以經由row operation變成I，表示他是可逆的 p.s可經由同樣順序的運算將一個I變成該矩陣的反矩陣

determinant (行列式) :

  1. 行列式性質 :

    • det(A) = det(A^T)
    • det(A*B) = det(A)*det(B)
    • det(A^-1) = 1/(det(A))
    • det(A+B) != det(A)+det(B)
    • det(kA) = (k)^n*det(A) k為一個常數，n表示A的階數
    • 上/下三角矩陣的行列式值為該矩陣的對角線元素合
    • 對一個矩陣的行或列乘k倍，其行列式值變為k倍
    • 將矩陣的兩行/列互換，其行列式值變為原本的-1倍
    • 將原本矩陣的一行乘上k倍加到另外一行，其行列式值不變
    • 矩陣任兩列/行程比例，其行列式值為0
    • 矩陣某行/列為0，其行列式值為0
    • A^-1 = adj(A)/det(A)
    • adj(A)表示一個陣列，其元素為以該元素為目標的其他餘子式展開和

  2. 其他重要操作 :

    • cofactor expansion (餘子式展開) :

      將矩陣以任一行/列展開，可便利算出行列式值
      tips : 以0較多的那一行/列展開可計算較少行列式
