Excel operations

  1. 一些常用操作 :

    1. 如果要從其他工作表、試算表引用內容要在引用的頁面按下enter
    2. 可以在檔案-選項-進階-自訂清單，來決定你需要的連續結果 ex.1月,2月...12月 或 早,中,晚 等等
    3. 可以用ctrl來多重選取資料
    4. 如果已知需要選取的儲存格範圍，可以使用 : 來省略選擇過程 ex. A2:D3 等等
    5. 對數字使用ctrl + 滑鼠拉伸會自動連加，否則只會複製同一個值
    6. 對於圖片的許多操作可以針對不同的區塊做
    7. 若圖表中的xy軸出現問題，從變更圖表類型去選擇新的圖表


  2. 快捷鍵區 :

    1. 可以使用F4來鎖定函數內的參數，讓之後對此函數的複製的該參數不會隨著平移
    2. alt + enter : 在儲存格內換行
    3. ctrl + home : 回到excel A1的儲存格
    4. ctrl + shift + 方向鍵 : 到儲存資料的邊界


  3. 常用區 :

    1. 跨欄至中 : 同資料在表格內統一顯示
    2. 設定格式化的條件 : 對於選擇的資料建立醒目的篩選
    3. 自動加總 == sum() : 加總選擇range中的所有數值
    4. 改變儲存格格式 : 可以改變數值的格式，用來以不同形態表示變數內容
    5. 填滿-數列 : 可以選擇要填滿的格數跟數值，等差數列或等比數列，需要知道資料總共有多少個
    6. 複製格式 : 用來將一個儲存格格式複製給其他儲存格，如果要複製欄列寬度必須把整欄、列一起選取
    7. 通用形式底下可以對所選數值做更進一步的調整
    8. 數值調整小數點位數


  4. 插入區 :

    1. 建議圖表 : 可以根據你在excel做的資料建立相對應的圖表，不能亂建立xy軸
    2. 樞紐分析 : 可以建立所需表單針對不同欄列的統計結果，東西有夠多

  6. 公式區 :

    1. 自動計算 : 將此工作表的所有函數重新計算，得到新結果


  7. 資料區 :

    1. 資料驗證 : 可以設定選擇範圍內的資料只能使用設定的參數
    2. 篩選 : 可以只顯示出選擇的標籤結果
    3. 檔案-選項-增益集-分析工具 : 會在資料區加上資料分析的部分
    4. 資料分析-敘述統計 : 可以列出所選擇資料的基本統計數字
    5. 資料剖析 : 用來把長串資料進行切割
    6. 資料分析-亂數產生器(均等分布) : 第一個參數是橫行個數與直行個數，均等分配表示會填入浮點數(精確到小數點第13位)
    7. 資料分析-等級和百分比(只接受一欄的資料(直行)) : 將所選資料排序，顯示PR值與原順序點
    8. 資料分析-亂數產生器(二項分配) : 給予成功機率與執行次數來計算1出現的次數
    9. 資料分析-亂數產生器(白努力分配) : 給予成功機率來計算答案是否是1出現的次數
    10. 資料分析-亂數產生器(常態分配) : 給予標準差與平均數，產生相關數組


  8. 校閱區 :



  9. 檢視區 :

    1. 凍結窗格 : 對於選擇上方的欄或列進行凍結，在離開可是範圍的時候可以持續可見


  10. 常用函數 :

    1. countif()函數-第一個參數是range，第二個參數是篩選目標，結果是統計後符合條件的總數
    2. round()函數 : 將所選數字四捨五入到小數點第n位，第一個參數是數字來源，第二個參數是需要round第幾位
    3. floor()函數 : 將指定數字取floor，後面放1
    4. rank.avg()函數 : 將所選的資料範圍進行排名(default是從大排到小)，參數分別是所選數字、比較的範圍，以及布林值(0表示從大排到小)
    5. rand()函數 : 產生一個亂數，調整亂數x範圍到 a<=x<=b (rand()*(a-b)+b)
    6. RANDBETWEEN()函數 : 產生一個在指定區間內的亂數，需要兩個引數，分別是上限及下限
