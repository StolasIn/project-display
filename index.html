應用統計excel筆記
    1. 一些常用操作 :
      1. 如果要從其他工作表、試算表引用內容要在引用的頁面按下enter
      2. 可以在檔案-選項-進階-自訂清單，來決定你需要的連續結果 ex.1月,2月...12月 或 早,中,晚 等等
      3. 可以用ctrl來多重選取資料
      4. 如果已知需要選取的儲存格範圍，可以使用 : 來省略選擇過程 ex. A2:D3 等等
      5. 對數字使用ctrl + 滑鼠拉伸會自動連加，否則只會複製同一個值


    2. 快捷鍵區 :
      1. 可以使用F4來鎖定函數內的參數，讓之後對此函數的複製的該參數不會隨著平移
      2. alt + enter : 在儲存格內換行
      3. ctrl + home : 回到excel A1的儲存格
      4. ctrl + shift + 方向鍵 : 到儲存資料的邊界


    3. 常用區 :
      1. 跨欄至中 : 同資料在表格內統一顯示
      2. 設定格式化的條件 : 對於選擇的資料建立醒目的篩選
      3. 自動加總 == sum() : 加總選擇range中的所有數值
      4. 改變儲存格格式 : 可以改變數值的格式，用來以不同形態表示變數內容
      5. 填滿 : 可以選擇要填滿的格數跟數值，等差數列或等比數列的
      6. 複製格式 : 用來將一個儲存格格式複製給其他儲存格，如果要複製欄列寬度必須把整欄、列一起選取


    4. 插入區 :
      1. 建議圖表 : 可以根據你在excel做的資料建立相對應的圖表，不能亂建立xy軸

    6. 公式區 :
      1. 自動計算 : 將此工作表的所有函數重新計算，得到新結果


    7. 資料區 :
      1. 資料驗證 : 可以設定選擇範圍內的資料只能使用設定的參數
      2. 篩選 : 可以只顯示出選擇的標籤結果

    主要STL介紹 :

    1. stack
    2. queue
    3. linked_list
    4. tree


    1. stack :




      • stack特徵 :
      • stack是一種LIFO的資料結構，函數呼叫疊代的時候就是一種stack的表現
      • stack可以使用linked_list實作，需要一個變數紀錄最頂端的元素



    2. queue :




      • queue特徵 :
      • queueg是一種FIFO的資料結構
      • stack可以使用linked_list實作，需要兩個變數分別紀錄最頭的元素與最尾巴的元素



    3. linked_list :




      • linked_list特徵 :
      • linked_list的每個節點通常由兩個元素(data與指向下一個節點的指標)組成(double linked_list有三個)
      • linked_list的插入與刪除都是O(1) (不考慮尋找點的過程)
      • linked_list不像array一樣有random access memory，所以linked_list的traverse需要 O(n)
      • 比起array適合用在頻繁做插入刪除且各數不多的資料



    4. tree :




      • tree特徵 :
      • 樹會有一個根節點
      • 樹的父節點與子節點會有parent child的關係
      • level被稱為樹高，是從根結點到最遠葉節點的距離
